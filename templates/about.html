My Projects

Can I Eat This?

Helps pregnant women quickly figure out if something is safe to eat.

Technology Used

  • Django
  • Elasticsearch as the primary data store
  • jQuery
  • Redis as a task queue (holds onto missed search terms) and to hold recent searches
  • Scraped information and indexed into Elasticsearch
  • Deployed using Gunicorn (monitored by Supervisor) and Nginx

Parks and Rec Quiz

A single page application (SPA) game that tests users' knowledge of the popular TV show, Parks and Recreation. Allows users to compare results via a leaderboard, share their scores on Facebook, and submit new questions.

Technology Used

  • Django REST Framework
  • AngularJS and jQuery
  • PostgreSQL
  • Redis (as a task queue and to hold leaderboard information)
  • Deployed using Gunicorn (monitored by Supervisor) and Nginx
  • Implemented Facebook's API

Tomatoflix

An interactive website that allows users to select random movies that are available to stream from Netflix. Users are able to search for movies by Rotten Tomatoes' ratings, Netflix rating, by movie or TV show, or genre.

Technology Used

  • Django
  • jQuery
  • PostgreSQL
  • Deployed using Gunicorn (monitored by Supervisor) and Nginx
  • Scraped data from Netflix (private API) and Rotten Tomatoes (public API... just thought it'd be fun) via Python scripting. Scraped detailed information for over 6,000 movie titles.
